## O que é ETL?

👉 ETL é o caminho que o dado percorre até se tornar informação útil.

Note:

- **O obvio precisa ser dito**.
- Integrar dados de diferentes fontes para aprimorar a **tomada de decisão**.
- Um processo mecânico.
- Não envolve análise de dados.
- Uma sigla, um [acrónimo](https://pt.wikipedia.org/wiki/Acr%C3%B3nimo):
  - **Extract (Extrair)**: Coletar dados das fontes.
  - **Transform (Transformar)**: Tratar, padronizar e enriquecer os dados.
  - **Load (Carregar)**: Armazenar os dados prontos em um destino final (análise de dados e tomada de decisão).

<!--v-->

## O que é ETL?

Por que ETL é necessário?

Note:

- Dados quase sempre estão espalhados em múltiplas fontes.
- Dados nascem para um contexto **operacional**, não analítico ([OLTP x OLAP](https://stackoverflow.com/questions/21900185/what-are-oltp-and-olap-and-what-is-the-difference-between-them)):
  - OLTP: On-line Transaction Processing.
  - OLAP: On-line Analytical Processing.
- Tempo excessivo gasto "arrumando dados" manualmente fazendo cada nova análise vira um retrabalho.
- Relatórios que não batem entre si.

## Extract

Tirar os dados de onde eles vivem.

Note:

- Acessar as fontes corretas.
- Extrair os dados necessários.
- Os dados são lidos, não alterados.
- Garantir que a extração seja confiável e repetível.

<!--v-->

## Extract

💡 Em ambientes reais, raramente existe uma única fonte.

Note:

- Bancos de dados relacionais (PostgreSQL, Oracle, MySQL).
- Planilhas (Excel, CSV).
- APIs (sistemas externos, serviços web).
- Arquivos de texto ou logs.
- SISOR, SIAFI, SIAD, Excel, CSV.

<!--v-->

## Extract

O que NÃO é responsabilidade do Extract?

Note:

- Corrigir erros de dados (Transform).
- Padronizar formatos (Transform).
- Criar métricas (Transform).
- Regras de negócio complexas (Transform).

from frictionless import Package

package = Package("datapackage.yaml")
vogais_df = package.get_resource("vowels").to_pandas()
consoantes_df = package.get_resource("consonants").to_pandas()

# https://chatgpt.com/share/697270fd-18dc-8003-b014-700e5c6e9322
df = pd.concat([consoantes_df, vogais_df], ignore_index=True)
df = df.sort_values("letter").reset_index(drop=True)
df["id"] = df.index + 1
df.to_csv("data/all_letters.csv", index=False)
